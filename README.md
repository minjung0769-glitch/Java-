# 11강 변수 선언과 초기화
- 변수선언: 데이터를 보관할 수 있는 공간을 만들고, 그곳에 이름을 부여
- int c, d; 여러개도 가능
- 초기회: 빈 변수에 값을 처음 넣음
- int a = 0; 변수 선언과 초기화를 한번에
- int c = 3, d = 4;: 여러 변수 한번에도 가능
- 초기화를 하지 않고 출력 시 컴파일 에러
